  1. Forklift Drivers: Insert data into each field and press enter to advance to the next field. Once the Bol data is inserted and enter is pressed, the data is submitted and the form resets, ready for the next set of data. A Bol is created and sent to the truck driver queue.

  2. Truck Drivers: Bols needed to be approved show up under the "Driver Approval" page. When clicked, the user is taken to a page where they can view, add, and edit all the data on the Bol. From here, the user can approve the bol and download it as a pdf. Approved bols are sent to a queue for managers to approve.

  3. Managers: Bols needed to be approved show up under the "Manager Approval" page. When clicked, the user is taken to a page where they can view, add, and edit all the data on the Bol. From here, the user can approve the bol or reject the bol. Approved bols are removed from the manager's queue while rejected bols are sent back to the truck driver queue.
